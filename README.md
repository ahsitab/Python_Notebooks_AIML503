# AIML 505: Python Fundamentals (Lecture 2)

Welcome to the repository for **AIML 505 - Lecture 2**. This repository features a comprehensive, hands-on Jupyter Notebook (`aiml505-lecture-2.ipynb`) designed to master the foundational concepts of Python programming, specifically focusing on control flow, loops, functions, and core data structures.

Whether you are building a baseline for machine learning or refining your Python programming fundamentals, this guide provides clear explanations accompanied by executable code examples.

---

## 📚 Key Topics Covered

### 1. Control Flow & Decision Making
* **Conditional Statements:** Practical usage of `if`, `elif`, and `else` ladders.
* **Operators:** Combining complex conditions with comparison and logical operators (`and`, `or`, `not`).
* **Advanced Decision Structures:** Working with nested conditional blocks and clean, single-line ternary expressions.

### 2. Loops & Repetition
* **`for` Loops:** Iterating over sequential data structures and leveraging `range()` parameters.
* **`while` Loops:** Constructing condition-based iterations and implementing user-input evaluation cycles.
* **Multi-Dimensional Iteration:** Nested loops for matrix scanning and visual pattern generation.
* **Loop Control Mechanics:** Precise flow manipulation using `break`, `continue`, and loop `else` blocks.

### 3. Functions & Scope
* **Definitions & Calls:** Standard structural practices for writing reusable Python components.
* **Dynamic Arguments:** Using positional, keyword, and default values alongside flexible `*args` and `**kwargs` parameters.
* **Variable Scope:** Navigating differences between `local`, `global`, and `nonlocal` environments.
* **Lambda Functions:** Creating inline, anonymous functions for compact computations.

### 4. Built-in Data Structures
* **Lists:** Slicing, item assignments, indexing, and comprehensive utility methods.
* **Tuples:** Understanding immutable sequence types for storing static, structured records.
* **Dictionaries:** Handling unordered key-value pairs for highly efficient record queries.
* **Sets:** Utilizing unique elements alongside traditional set operations (unions, intersections, differences).

### 5. Advanced Techniques & Data Manipulation
* **Comprehensions:** Elegant, readable single-line list and dictionary comprehension expressions.
* **Data Synthesis:** Practical exercises bringing all these foundational concepts together for complex problem-solving.
